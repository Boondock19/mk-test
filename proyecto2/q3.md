# Reporte de pasajeros con sus bookings

```
SELECT
	t.passenger_name,
	SUM(b.total_amount) total_amount,
	COUNT(DISTINCT(b.book_ref)) count_bookings
FROM 
	tickets t
JOIN 
	bookings b ON b.book_ref = t.book_ref
GROUP BY t.passenger_name;
```
